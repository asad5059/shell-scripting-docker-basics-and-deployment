# 🐚 Introduction to Bash Scripting

## 📌 1. What is Shell & Bash?

### 🖥️ GUI vs CLI

* **GUI (Graphical User Interface)**: Interaction through visual elements like buttons and menus.
* **CLI (Command Line Interface)**: Interaction through text-based commands.

👉 The CLI is **faster and more powerful** for repetitive and automation tasks compared to GUI.

---

### 🐚 The Shell

* The **Shell** is a program in Linux that:

  * Interprets user commands
  * Executes them by communicating with the system
  * Shell Scripting and Bash Scripting sometimes used interchangeably.

---

### 💻 Bash (Bourne Again Shell)

* Bash is the **most widely used shell** in Linux.
* It is not just a command runner — it is a **full-fledged scripting language**.
* Used for:

  * Automation
  * Task scheduling
  * System operations

---

### 🖲️ Terminal

* The **Terminal** is the interface (window/application) where:

  * You type commands
  * Run scripts
  * Interact with the shell

---
